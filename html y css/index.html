Formularios

Para crear formulario en html utilizamos la etiqueta form, la cual posee una etiqueta de apertura y de cierre. Dentro del form tenemos las etiquetas input, las cuales sirven para recibir un dato del usuario. Dentro de los input podemos poner unn texto que le indique al usuario lo que debe poner en el campo correspondiente, esto lo hacemos mediante un atributo que se llama placeholder al cual le podemos asignar el texto que querramos. Este texto desaparecera cuando el usuario quera escribir algo en el formulario. Mediante el atributo required hacemos que el usuario no pueda enviar el formulario sin completar todas las casillas requeridas.
