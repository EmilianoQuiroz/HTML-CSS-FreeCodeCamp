Css: Cascading Style Sheets.

Hoja de estilos en cascada.

Mientras que en HTML definimos la estructura de nuestra pagina web en CSS podemos dedfinir la apariencia de la misma. CSS es un lenguaje basado en reglas, estas reglas determinaran el estilo que se aplicara a un elemento o a un grupo de elementos.

Selectores en css

Los selectores en css sirven para seleccionar el tipo de elemento al que se le va a aplicar el estilo. Unos ejemplos de esto pueden ser:
h1 { color: blue; }
Esto hace que todos los elementos de tipo h1 tengam un color azul.
